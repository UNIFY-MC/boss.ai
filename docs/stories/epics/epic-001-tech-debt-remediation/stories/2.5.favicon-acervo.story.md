## Goal

Criar um favicon único para o "Acervo" (não da marca de uma formação específica) em `_shared/favicon.svg`, carregado por todas as 11 páginas. Hoje só o Build Day tem favicon (43 KB PNG); o resto carrega o default do browser.

## Acceptance Criteria

- [ ] AC1: Existe `_shared/favicon.svg` com identidade "Acervo" (não uma marca de formação)
- [ ] AC2: SVG < 2 KB, usa `currentColor` ou cor neutra que funcione em light + dark mode
- [ ] AC3: Em todas as 11 páginas, `<link rel="icon" type="image/svg+xml" href="/_shared/favicon.svg">` (ou path relativo correcto)
- [ ] AC4: PNG existente do Build Day (43 KB) optimizado ou removido (substituído pelo SVG partilhado)
- [ ] AC5: Smoke test em browser: aba do browser mostra o favicon em 11/11 páginas

## Tasks

- [ ] Task 1: Design do favicon "Acervo" (símbolo simples, abstracto, coerente com identidade do acervo) — pode ser delegado/discutido com Mário

- [ ] Task 4: Adicionar `<link rel="icon">` em 11/11 páginas
- [ ] Task 5: Remover ou substituir favicon PNG do Build Day
