## Goal

Memory files do Claude (em `C:\Users\mario\.claude\projects\...\memory\`) crescem ao longo do tempo. O `MEMORY.md` tem cap de 200 linhas — quando exceder, há **truncagem silenciosa**. Hoje o tamanho total é ~4 KB (irrelevante), mas em 6-12 meses pode degradar.

Criar uma política explícita: `memory-lifecycle.md` em `.claude/rules/` que define:
- Threshold para compactação de memórias
- Lifecycle de memórias `project` (archive após 60 dias sem update)
- Audit de slugs duplicados
- Comando ou processo manual para compactação
