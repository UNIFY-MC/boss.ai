## Goal

`core-config.yaml` tem flag `github-copilot: enabled` mas a directoria `.github/agents/` não existe. Resultado: utilizadores do Copilot ficam sem agentes AIOX.

Decisão: (a) materializar `.github/agents/` via `aiox ideSync` para incluir o Copilot na lista de IDEs sincronizadas, OU (b) desactivar a flag (`github-copilot: disabled` ou remover) se não há intenção de suportar Copilot agora.
