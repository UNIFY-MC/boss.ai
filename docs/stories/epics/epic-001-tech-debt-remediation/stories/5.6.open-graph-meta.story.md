# Story 5.6: Open Graph / Twitter Cards meta tags

**Epic:** epic-001-tech-debt-remediation
**Status:** Draft
**Wave:** 5
**TD source:** TD-022
**Severity:** LOW (mas polish significativo — previews ricos em Slack/WhatsApp/Notion)
**Effort:** S (1-2h)

## Goal

Adicionar Open Graph + Twitter Cards meta tags em todas as 11 páginas do acervo, mais 1 imagem OG genérica em `_shared/og-image.png` (1200×630). Previews em Slack/WhatsApp/Notion passam de vazios a ricos.

## Acceptance Criteria

- [ ] AC1: Em todas as 11 páginas, existem meta tags:
  - `<meta property="og:title">` com título da página
  - `<meta property="og:description">` com descrição curta
  - `<meta property="og:image">` apontando para imagem OG (genérica do acervo OU específica da formação se existir)
  - `<meta property="og:url">`
  - `<meta property="og:type" content="website">`
  - Twitter Card equivalentes: `twitter:card`, `twitter:title`, `twitter:description`, `twitter:image`
- [ ] AC2: Existe `_shared/og-image.png` (ou `.svg` convertido para `.png`) de 1200×630 px representando o "Acervo"
- [ ] AC3: Cada formação pode opcionalmente override com OG image própria (estrutura permite isto)
- [ ] AC4: Teste em https://www.opengraph.xyz (ou equivalente): 11/11 páginas mostram preview rico
- [ ] AC5: Teste no Slack/WhatsApp em ambiente real: preview aparece coerente

## Tasks

- [ ] Task 1: Design da imagem OG genérica (texto "Acervo de Formações" + identidade visual)
- [ ] Task 2: Exportar como 1200×630 PNG em `_shared/og-image.png`
- [ ] Task 3: Definir template de meta tags
- [ ] Task 4: Aplicar em 11/11 páginas (com título e descrição apropriados a cada)
- [ ] Task 5: Validar com opengraph.xyz
- [ ] Task 6: Teste real em Slack/WhatsApp
