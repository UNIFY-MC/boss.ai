# Story 2.6: Audit pesos Geist + preload hero font + `font-display: swap`

**Epic:** epic-001-tech-debt-remediation
**Status:** Draft
**Wave:** 2
**TD source:** TD-014 (reframe de Uma — Phase 6)
**Severity:** MEDIUM
**Effort:** S (1h)

## Goal

Reduzir payload combinado de fontes e eliminar FOIT, **sem reduzir o número de famílias** (as 6 famílias diferentes são valor arquivístico — cada formação tem a sua tipografia). Foco em:
1. Audit dos pesos do Geist actualmente carregados (3-9) e cortar os não usados (alvo: 4-5 pesos)
2. `preload` da hero font de cada página
3. `font-display: swap` consistente em todas as `@font-face` / Google Fonts URLs

## Acceptance Criteria

- [ ] AC1: Audit completo: em cada página, listar quais pesos de cada família são **efectivamente** usados (via inspecção de CSS computado ou Chrome Coverage)
- [ ] AC2: Reduzir Geist a 4-5 pesos (e.g., 400, 500, 600, 700 — manter apenas os que são usados)
- [ ] AC3: Outras famílias auditadas e cortadas pesos mortos onde aplicável
- [ ] AC4: Cada página tem `<link rel="preload" as="font" type="font/woff2" crossorigin href="...">` para a **hero font** (font usada no LCP candidate — geralmente o h1 principal)
- [ ] AC5: Todos os `<link>` para Google Fonts incluem `&display=swap`; todos os `@font-face` self-hosted incluem `font-display: swap`
- [ ] AC6: Network panel: FOIT visível = 0 (texto rende com fallback até web font carregar)
- [ ] AC7: Lighthouse não flagga "Ensure text remains visible during webfont load"
- [ ] AC8: Payload combinado de fontes reduzido em pelo menos 30% face ao estado actual

## Tasks

- [ ] Task 1: Em cada uma das 11 páginas, identificar a hero font (font do primeiro `<h1>` ou maior elemento visual)
- [ ] Task 2: Chrome Coverage tool ou inspecção manual: listar pesos efectivamente usados de cada família
- [ ] Task 3: Reduzir URLs do Google Fonts para pedir apenas pesos usados
- [ ] Task 4: Adicionar `<link rel="preload">` para hero font em cada uma das 11 páginas
- [ ] Task 5: Adicionar `&display=swap` em todos os URLs do Google Fonts; `font-display: swap` em todos os `@font-face` self-hosted
- [ ] Task 6: Medir payload antes/depois e validar redução
- [ ] Task 7: Smoke test visual em 11/11 páginas: tipografia idêntica (apenas pesos não usados removidos)

## Out of scope

- Reduzir famílias (6 → menos) — explicitamente fora do scope (valor arquivístico)
- Self-host de fontes do Google (separado se desejado)
