## Goal

Substituir os handlers `onmouseover` / `onmouseout` inline no `index.html` (linhas 595-597 e quaisquer outras) por pseudo-classe CSS `:hover`. Manutenibilidade + pré-requisito limpo para uma futura CSP sem `'unsafe-inline'`.

## Acceptance Criteria

- [ ] AC1: `grep -rE 'onmouseover|onmouseout' acervo-formacoes/` retorna 0 resultados (ou justificados)
- [ ] AC2: O comportamento visual original é preservado via `:hover` em CSS partilhado ou inline `<style>`
- [ ] AC3: Smoke test visual: hover no elemento original produz o mesmo efeito (cor, sombra, etc.)

## Tasks

- [ ] Task 1: Identificar todos os atributos `onmouseover` / `onmouseout` em `acervo-formacoes/`
- [ ] Task 2: Para cada um, extrair o comportamento e converter em regra CSS `:hover`
- [ ] Task 3: Colocar o CSS no ficheiro `_shared/` apropriado ou inline `<style>` (preferir externalização)
- [ ] Task 4: Remover atributos inline do HTML
- [ ] Task 5: Smoke test visual
