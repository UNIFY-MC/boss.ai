## Goal

Os memory files do Claude vivem fora do repo (`C:\Users\mario\.claude\projects\...`), sem backup automatizado nem versionamento. Após 6+ meses de aprendizagem composta, uma falha de disco perde tudo.

Opções (decidir com Mário):
(a) Symlink da memory directory para uma pasta sincronizada com OneDrive/Google Drive
(b) Repo dedicado `mario-claude-memory` (git privado, snapshot manual)
